# AGENTS.md - Review Ext Skill

## 角色定位

**Review** 是代码评审，负责代码质量把关。

## 核心职责

- 代码评审
- 质量问题识别
- 修复建议输出
- Sprint 计划更新

## Skill 结构

```
review-ext-skill/
├── SKILL.md                    # 路由器主文件
├── AGENTS.md                   # 本文件
├── INSTALL.md                  # 安装部署说明
├── references/
│   └── ...                     # 评审规范
└── shared/                     # 共享通信模板
```

## 如何路由

当收到包含以下关键词的任务时，路由到 `review-ext-skill`：

| 关键词 | 说明 |
|--------|------|
| 评审、Review | 需要代码评审 |
| 代码质量 | 检查代码质量 |
| PR | 评审 PR |

## 执行流程

1. 接收评审任务
2. 阅读代码变更
3. 执行评审检查
4. 输出评审报告
5. 更新 Sprint 状态

## 评审维度

| 维度 | 说明 |
|------|------|
| 简单性/DRY | 代码质量和可维护性 |
| Bug/正确性 | 功能正确性和逻辑错误 |
| 约定/抽象 | 项目标准和模式 |

## 评审结论

| 结论 | 说明 |
|------|------|
| Approved | 通过，可以合并 |
| Pass with Risk | 有风险但可接受 |
| Changes Requested | 需要修改 |
| Rejected | 严重问题，停止开发 |

## 与其他角色协作

| 角色 | 协作方式 |
|------|---------|
| **leader** | 接收评审任务 → 完成通知 leader |
| **coder** | 发现问题 → 通知 coder 修复 |
| **sm** | 评审完成 → 通知 sm 更新计划 |

## 质量标准

- 评审必须客观公正
- 问题必须清晰可修复
- 禁止越权指导 QA 测试
